# react-practice-demo

## 安装

```
  # 安装依赖
  $ npm install

```

**运行项目**

```bash
  # 编译代码
  $ npm run build

  # 查看效果
  $ http://127.0.0.1:8080

```

## 使用方式

 ES6/7 :  修改src/ES6/来测试你的ES6/7语法即可

 React :  修改src/entry.js来学习react代码

 Redux :  修改src/redux/来学习redux代码

 async/generator : 如果你想用更高级的类似async/Promise等等语法，请修改webpack.config.js
